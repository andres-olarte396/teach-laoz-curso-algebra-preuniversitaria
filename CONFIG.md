# Configuración del Curso de Álgebra Pre-universitaria

## Parámetros del Curso

- **Tema**: Álgebra para Pre-universitarios
- **Complejidad**: Alta
- **Duración**: 50 horas
- **Audiencia**: Estudiantes de último año de secundaria preparándose para exámenes de admisión universitaria
- **Prerrequisitos**: Aritmética básica, operaciones con fracciones, conceptos básicos de geometría
