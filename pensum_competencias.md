# PENSUM DE COMPETENCIAS: ÁLGEBRA PARA PRE-UNIVERSITARIOS

## PERFIL DE EGRESO

Al finalizar este curso, el estudiante será capaz de:

1. **Manipular expresiones algebraicas con fluidez**: Simplificar, expandir y factorizar expresiones algebraicas complejas aplicando productos notables, propiedades de exponentes y técnicas de factorización avanzadas.

2. **Resolver ecuaciones y sistemas de ecuaciones**: Aplicar múltiples métodos (algebraicos, gráficos, matriciales) para resolver ecuaciones lineales, cuadráticas, exponenciales, logarítmicas y sistemas de ecuaciones de hasta 3x3.

3. **Analizar y graficar funciones matemáticas**: Determinar dominio, rango, asíntotas, extremos y comportamiento de funciones polinomiales, racionales, radicales, exponenciales y logarítmicas, representándolas gráficamente con precisión.

4. **Modelar problemas del mundo real**: Traducir situaciones de física, economía, ingeniería y ciencias al lenguaje algebraico, construir modelos matemáticos apropiados y resolver problemas aplicados.

5. **Demostrar preparación para cálculo universitario**: Poseer las competencias algebraicas fundamentales requeridas para cursos universitarios de cálculo diferencial, integral y álgebra lineal.

6. **Razonar matemáticamente con rigor**: Justificar procedimientos, validar soluciones, identificar errores en razonamientos algebraicos y comunicar ideas matemáticas con claridad y precisión.

## COMPETENCIAS TRANSVERSALES

Además de las competencias específicas por módulo, el curso desarrolla las siguientes competencias transversales:

### 1. Pensamiento Crítico y Analítico

- Evaluar la validez de soluciones algebraicas
- Identificar errores comunes en procedimientos matemáticos
- Seleccionar estrategias de resolución apropiadas según el contexto

### 2. Resolución de Problemas

- Descomponer problemas complejos en pasos manejables
- Aplicar múltiples enfoques para verificar soluciones
- Transferir conocimientos algebraicos a contextos nuevos

### 3. Comunicación Matemática

- Expresar ideas matemáticas con notación precisa
- Justificar procedimientos y conclusiones
- Interpretar y explicar resultados en contexto

### 4. Uso de Tecnología

- Emplear calculadoras científicas eficientemente
- Utilizar software de graficación (GeoGebra, Desmos)
- Verificar resultados algebraicos con herramientas digitales

### 5. Autonomía en el Aprendizaje

- Identificar fortalezas y debilidades propias
- Buscar recursos adicionales para reforzar conceptos
- Desarrollar estrategias de estudio efectivas

## NIVELES DE DOMINIO

Para cada competencia, se definen tres niveles de dominio:

### Nivel Básico (60-74%)

- Ejecuta procedimientos algebraicos estándar con guía
- Resuelve problemas directos de aplicación
- Identifica conceptos fundamentales

### Nivel Intermedio (75-89%)

- Aplica técnicas algebraicas de manera autónoma
- Resuelve problemas que requieren múltiples pasos
- Conecta conceptos entre diferentes módulos

### Nivel Avanzado (90-100%)

- Selecciona estrategias óptimas para problemas complejos
- Resuelve problemas no rutinarios creativamente
- Generaliza conceptos a situaciones nuevas
- Explica y justifica procedimientos con rigor
