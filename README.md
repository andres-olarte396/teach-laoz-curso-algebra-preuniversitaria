## 📋 Descripción

Curso completo de **Álgebra Preuniversitaria** diseñado para estudiantes de último año de secundaria que se preparan para exámenes de admisión universitaria en áreas de Ciencias, Ingeniería y Economía. Este material educativo forma parte del proyecto **Teach LaOz Learning Management System** y está optimizado para el autoaprendizaje con evaluaciones continuas.

## 🎯 Objetivos del Curso

1. **Dominar el lenguaje algebraico**: Traducir problemas del lenguaje natural al algebraico y viceversa
2. **Resolver ecuaciones y sistemas complejos**: Aplicar múltiples métodos algebraicos, gráficos y numéricos
3. **Analizar y graficar funciones**: Identificar características clave y representarlas con precisión
4. **Aplicar álgebra a problemas reales**: Modelar situaciones de física, economía, ingeniería y ciencias
5. **Prepararse para cálculo**: Desarrollar competencias fundamentales para cursos universitarios

## 📚 Estructura del Curso

### Información General

- **Complejidad**: Alta
- **Duración**: 50 horas (10 semanas)
- **Carga Semanal**: 5 horas
- **Modalidad**: Autoestudio con evaluaciones semanales
- **Ritmo**: Intensivo

### Prerrequisitos

✅ Aritmética básica (operaciones con números enteros, decimales y fracciones)  
✅ Operaciones con fracciones (suma, resta, multiplicación, división, simplificación)  
✅ Conceptos básicos de geometría (áreas, perímetros, teorema de Pitágoras)  
✅ Manejo de calculadora científica

## 📖 Contenido del Curso

El curso está organizado en **13 módulos principales**:

### 📍 Módulo 0: Diagnóstico y Nivelación (3h)
- Repaso de aritmética
- Introducción al lenguaje algebraico
- Evaluación diagnóstica

### 📍 Módulo 1: Expresiones Algebraicas (5h)
- Operaciones con expresiones algebraicas
- Productos notables
- División de polinomios

### 📍 Módulo 2: Factorización (6h)
- Técnicas básicas de factorización
- Factorización de trinomios
- Casos especiales de factorización

### 📍 Módulo 3: Ecuaciones Lineales (5h)
- Resolución de ecuaciones lineales
- Ecuaciones con fracciones y decimales
- Problemas de aplicación

### 📍 Módulo 4: Sistemas de Ecuaciones (5h)
- Métodos de resolución (sustitución, igualación, eliminación)
- Sistemas de tres variables
- Aplicaciones de sistemas

### 📍 Módulo 5: Ecuaciones Cuadráticas (4h)
- Métodos de resolución
- Naturaleza de las raíces

### 📍 Módulo 6: Desigualdades (5h)
- Desigualdades lineales
- Desigualdades cuadráticas y racionales
- Valor absoluto

### 📍 Módulo 7: Funciones (4h)
- Conceptos fundamentales
- Operaciones con funciones
- Función inversa

### 📍 Módulo 8: Funciones Polinomiales (4h)
- Funciones cuadráticas y cúbicas
- Teorema del residuo y del factor

### 📍 Módulo 9: Funciones Racionales (5h)
- Dominio y asíntotas
- Gráficas de funciones racionales

### 📍 Módulo 10: Funciones Exponenciales y Logarítmicas (4h)
- Propiedades y gráficas
- Ecuaciones exponenciales y logarítmicas

### 📍 Módulo 11: Sucesiones y Series (4h)
- Progresiones aritméticas y geométricas
- Suma de series

### 📍 Módulo 12: Matrices y Determinantes (2h)
- Operaciones con matrices
- Resolución de sistemas con matrices

## 🎓 Metodología de Aprendizaje

Cada subtema incluye:

1. **📝 Contenido Teórico** (`*_contenido.md`): Explicaciones detalladas con ejemplos
2. **✏️ Ejercicios Prácticos** (`*_ejercicios.md`): Problemas graduados por dificultad
3. **📊 Evaluaciones** (`*_evaluacion.md`): Preguntas de autoevaluación
4. **🎬 Guiones de Video** (`*_guion.md`): Scripts para videos educativos

## 📊 Sistema de Evaluación

- **Evaluaciones Diagnósticas**: Al inicio del curso
- **Evaluaciones Formativas**: Al final de cada subtema
- **Evaluaciones Sumativas**: Al final de cada módulo
- **Proyecto Final**: Integración de todos los conceptos

### Criterios de Aprobación

- **Mínimo para aprobar**: 70%
- **Asistencia/participación**: Recomendada
- **Ejercicios prácticos**: Obligatorios

## 🚀 Cómo Usar Este Material
