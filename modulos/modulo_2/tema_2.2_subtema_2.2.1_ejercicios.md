### Ejercicio 1
**Enunciado**: Verificar si $x^2 + 6x + 9$ es un trinomio cuadrado perfecto y factorizarlo

**Solución**:
```
VERIFICACION (3 preguntas):

Pregunta 1: ¿El primer término es un cuadrado perfecto?
  x² = (x)²  ✓
  a = x

Pregunta 2: ¿El tercer término es un cuadrado perfecto?
  9 = 3²  ✓
  b = 3

Pregunta 3: ¿El término medio es el doble producto?
  2ab = 2·x·3 = 6x  ✓

CONCLUSIÓN: SÍ es TCP

FACTORIZACIÓN:
  x² + 6x + 9 = (x + 3)²

VERIFICACIÓN:
  (x + 3)² = (x + 3)(x + 3) = x² + 3x + 3x + 9 = x² + 6x + 9 ✓

Respuesta: (x + 3)²
```

**Rúbrica**:
- Verificación de las 3 preguntas: 40%
- Factorización correcta: 40%
- Verificación final: 20%

### Ejercicio 2
**Enunciado**: Factorizar $m^2 - 10m + 25$

**Solución**:
```
VERIFICACIÓN TCP:

Pregunta 1: m² = (m)²  ✓  ⟹  a = m
Pregunta 2: 25 = 5²  ✓  ⟹  b = 5
Pregunta 3: 2ab = 2·m·5 = 10m  ✓ (coincide con coeficiente)

Signo: Como el término medio es NEGATIVO, el binomio es (a - b)

FACTORIZACIÓN:
  m² - 10m + 25 = (m - 5)²

Verificación:
  (m - 5)² = m² - 5m - 5m + 25 = m² - 10m + 25 ✓

Respuesta: (m - 5)²
```

**Rúbrica**:
- Identificación de TCP con signo negativo: 40%
- Factorización correcta: 40%
- Verificación: 20%

### Ejercicio 5
**Enunciado**: Factorizar $9a^2 - 24ab + 16b^2$

**Solución**:
```
VERIFICACIÓN TCP:

Pregunta 1: 9a² = (3a)²  ✓  ⟹  primer término = 3a
Pregunta 2: 16b² = (4b)²  ✓  ⟹  segundo término = 4b
Pregunta 3: 2·(3a)·(4b) = 24ab  ✓

Signo: Negativo en el medio ⟹ (a - b)

FACTORIZACIÓN:
  9a² - 24ab + 16b² = (3a - 4b)²

VERIFICACIÓN:
  (3a - 4b)² = 9a² - 12ab - 12ab + 16b² = 9a² - 24ab + 16b² ✓

Respuesta: (3a - 4b)²
```

**Rúbrica**:
- TCP con dos variables: 40%
- Identificación de raíces 3a y 4b: 35%
- Factorización correcta: 25%

**Solución**:
```

### Ejercicio 12
**Enunciado**: Resolver la ecuación $4x^2 - 12x + 9 = 0$ reconociendo el TCP

**Solución**:
```
PASO 1 - Reconocer TCP:
  4x² = (2x)²
  9 = 3²
  2·(2x)·3 = 12x  ✓
  
  Es TCP: 4x² - 12x + 9 = (2x - 3)²

PASO 2 - Resolver:
  (2x - 3)² = 0
  2x - 3 = 0  [raíz doble]
  2x = 3
  x = 3/2

VERIFICACIÓN:
  4(3/2)² - 12(3/2) + 9
  = 4·9/4 - 18 + 9
  = 9 - 18 + 9 = 0 ✓

NOTA: TCP siempre da raíz doble (multiplicidad 2)

Respuesta: x = 3/2 (raíz doble)
```

**Rúbrica**:
- Factorización como TCP: 35%
- Resolución correcta: 30%
- Reconocimiento de raíz doble: 20%
- Verificación: 15%

PASO 3 - Reescribir:
  (x² + y²)² - 4z²

PASO 4 - Reconocer diferencia de cuadrados:

  (x² + y² - 2z)(x² + y² + 2z)
  = (x² + y²)² - (2z)²
  = x⁴ + 2x²y² + y⁴ - 4z² ✓

Respuesta: (x² + y² - 2z)(x² + y² + 2z)
```
