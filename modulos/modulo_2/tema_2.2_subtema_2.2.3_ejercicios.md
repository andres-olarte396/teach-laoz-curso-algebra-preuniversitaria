## Nivel Básico (60-74%)

### Ejercicio 1
**Enunciado**: Factorizar $2x^2 + 5x + 3$

**Solución**:
```
MÉTODO AC (Agrupación):

PASO 1 - Calcular AC:
  A = 2, C = 3
  AC = 2×3 = 6

PASO 2 - Buscar dos números que:
  Producto = 6
  Suma = 5 (el coeficiente B)
  
  Opciones: 2 y 3  (2×3 = 6, 2+3 = 5) ✓

PASO 3 - Reescribir el término medio:
  2x² + 5x + 3 = 2x² + 2x + 3x + 3

PASO 4 - Agrupar:
  = (2x² + 2x) + (3x + 3)
  = 2x(x + 1) + 3(x + 1)
  = (x + 1)(2x + 3)

VERIFICACIÓN:
  (x + 1)(2x + 3) = 2x² + 3x + 2x + 3 = 2x² + 5x + 3 ✓

Respuesta: (x + 1)(2x + 3)
```

**Rúbrica**:
- Cálculo correcto de AC: 25%
- Identificación de los números: 30%
- Agrupación correcta: 30%
- Verificación: 15%

### Ejercicio 6
**Enunciado**: Factorizar $4x^2 + 12x + 9$

**Solución**:
```
OBSERVACIÓN: Verificar si es trinomio cuadrado perfecto

PASO 1 - Verificar TCP:
  ¿(2x)² = 4x²?  ✓
  ¿3² = 9?  ✓
  ¿2(2x)(3) = 12x?  ✓

CONCLUSIÓN: Es TCP
  4x² + 12x + 9 = (2x + 3)²

ALTERNATIVA con Método AC:
  AC = 4×9 = 36
  Números: 6 y 6 (producto = 36, suma = 12)
  = (2x + 3)(2x + 3) = (2x + 3)²

Respuesta: (2x + 3)²
```

### Ejercicio 13
**Enunciado**: Resolver la ecuación $6x^2 + x - 2 = 0$ factorizando

**Solución**:
```
PASO 1 - Factorizar:
  AC = 6×(-2) = -12
  Números: 4 y -3  (4×-3 = -12, 4-3 = 1)
  
  6x² + x - 2 = 6x² + 4x - 3x - 2
             = 2x(3x + 2) - 1(3x + 2)
             = (3x + 2)(2x - 1)

PASO 2 - Aplicar producto cero:
  (3x + 2)(2x - 1) = 0
  
  3x + 2 = 0  ⟹  x = -2/3
  2x - 1 = 0  ⟹  x = 1/2

VERIFICACIÓN para x = 1/2:
  6(1/2)² + (1/2) - 2 = 6/4 + 1/2 - 2 = 3/2 + 1/2 - 2 = 0 ✓

Respuesta: x = -2/3 o x = 1/2
```

**Rúbrica**:
- Factorización correcta: 40%
- Aplicación de producto cero: 30%
- Soluciones correctas: 20%
- Verificación: 10%

### Ejercicio 14
**Enunciado**: Factorizar $-3x^2 + 10x - 8$

**Solución**:
```
PASO 1 - Extraer el coeficiente negativo:
  -3x² + 10x - 8 = -(3x² - 10x + 8)

PASO 2 - Factorizar 3x² - 10x + 8:
  AC = 3×8 = 24
  Números: -4 y -6  (-4×-6 = 24, -4-6 = -10) ✓
  
  3x² - 10x + 8 = 3x² - 4x - 6x + 8
                = x(3x - 4) - 2(3x - 4)
                = (3x - 4)(x - 2)

PASO 3 - Resultado final:
  -3x² + 10x - 8 = -(3x - 4)(x - 2)

VERIFICACIÓN:
  -(3x - 4)(x - 2) = -(3x² - 6x - 4x + 8)
                   = -(3x² - 10x + 8)
                   = -3x² + 10x - 8 ✓

Respuesta: -(3x - 4)(x - 2) o (4 - 3x)(x - 2)
```

**Rúbrica**:
- Manejo del coeficiente principal negativo: 35%
- Factorización del trinomio: 35%
- Respuesta completa: 20%

**Solución**:
```

SUSTITUCIÓN: Sea u = ab
  6(ab)² + 7(ab) - 20 = 6u² + 7u - 20

FACTORIZAR:
  AC = 6×(-20) = -120
  Números: 15 y -8  (15×-8 = -120, 15-8 = 7) ✓
  
  6u² + 7u - 20 = 6u² + 15u - 8u - 20
                = 3u(2u + 5) - 4(2u + 5)
                = (2u + 5)(3u - 4)

REVERTIR SUSTITUCIÓN:
  = (2ab + 5)(3ab - 4)

VERIFICACIÓN:
  (2ab + 5)(3ab - 4)
  = 6a²b² - 8ab + 15ab - 20
  = 6a²b² + 7ab - 20 ✓

Respuesta: (2ab + 5)(3ab - 4)
```

**Rúbrica**:
- Reconocimiento del patrón con ab: 30%
