### Ejercicio 1
**Enunciado**: Factorizar $6x + 12$

**Solución**:
```
PASO 1 - Identificar el MCD:
  MCD(6, 12) = 6

PASO 2 - Dividir cada término:
  6x ÷ 6 = x
  12 ÷ 6 = 2

PASO 3 - Escribir factorización:
  6x + 12 = 6(x + 2)

VERIFICACIÓN:
  6(x + 2) = 6x + 12 ✓

Respuesta: 6(x + 2)
```

**Rúbrica**:
- Identificación correcta del MCD (6): 40%
- Factorización correcta: 40%
- Verificación: 20%

### Ejercicio 2
**Enunciado**: Factorizar $15x^2 - 10x$

**Solución**:
```
PASO 1 - MCD numérico: MCD(15, 10) = 5
PASO 2 - MCD literal: x (menor exponente)
PASO 3 - Factor común: 5x

Factorización:
  15x² - 10x = 5x(3x - 2)

Verificación:
  5x·3x - 5x·2 = 15x² - 10x ✓

Respuesta: 5x(3x - 2)
```

**Rúbrica**:
- MCD numérico correcto: 30%
- MCD literal correcto: 30%
- Factorización completa: 40%

### Ejercicio 3
**Enunciado**: Factorizar $8a^2b + 12ab^2$

**Solución**:
```
PASO 1 - MCD numérico: MCD(8, 12) = 4
PASO 2 - Variables comunes: a¹ y b¹
PASO 3 - Factor común total: 4ab

Factorización:
  8a²b + 12ab² = 4ab(2a + 3b)

Verificación:
  4ab·2a + 4ab·3b = 8a²b + 12ab² ✓

Respuesta: 4ab(2a + 3b)
```

**Rúbrica**:
- Identificación de factor común numérico: 25%
- Identificación de variables comunes: 25%
- Factorización correcta: 30%
- Verificación: 20%

### Ejercicio 4
**Enunciado**: Factorizar $-3x^2 - 9x$

**Solución**:
```
PASO 1 - Factor común: -3x (extraer el negativo)

Factorización:
  -3x² - 9x = -3x(x + 3)

Alternativa:
  También válido: 3x(-x - 3)
  Pero preferimos el coeficiente negativo fuera

Verificación:
  -3x·x - 3x·3 = -3x² - 9x ✓

Respuesta: -3x(x + 3)
```

**Rúbrica**:

- Factorización correcta: 40%

### Ejercicio 5
**Enunciado**: Factorizar $21m^3n^2 - 14m^2n^3 + 7mn$

**Solución**:
```
PASO 1 - MCD numérico: MCD(21, 14, 7) = 7
PASO 2 - Variables: m¹ (menor exponente) y n¹
PASO 3 - Factor común: 7mn

Factorización:
  21m³n² - 14m²n³ + 7mn = 7mn(3m²n - 2mn² + 1)

Verificación:
  7mn·3m²n = 21m³n²  ✓
  7mn·(-2mn²) = -14m²n³  ✓
  7mn·1 = 7mn  ✓

Respuesta: 7mn(3m²n - 2mn² + 1)
```

**Rúbrica**:
- MCD correcto (7mn): 30%

### Ejercicio 6
**Enunciado**: Factorizar $x(a + b) + y(a + b)$

**Solución**:
```
OBSERVACIÓN: (a + b) aparece en ambos términos

PASO 1 - Identificar factor común:
  Factor común: (a + b)

PASO 2 - Extraer:
  x(a + b) + y(a + b) = (a + b)(x + y)

VERIFICACIÓN:
  (a + b)(x + y) = x(a + b) + y(a + b) ✓

Respuesta: (a + b)(x + y)
```

**Rúbrica**:
- Identificación de expresión común: 35%
- Extracción correcta: 40%
- Verificación: 15%
- Notación adecuada: 10%

```

  Factor numérico: 3
  Factor expresión: (x - 2)
```

### Ejercicio 8
**Enunciado**: Factorizar $2a^2 - 8ab + 2ac - 8bc$

**Solución**:
```
PASO 1 - Factor común inicial:
  2a² - 8ab + 2ac - 8bc = 2(a² - 4ab + ac - 4bc)

PASO 2 - Agrupar términos dentro del paréntesis:
  = 2[a(a - 4b) + c(a - 4b)]

PASO 3 - Factor común (a - 4b):
  = 2(a - 4b)(a + c)

VERIFICACIÓN:
  2(a - 4b)(a + c)
  = 2[(a - 4b)·a + (a - 4b)·c]
  = 2[a² - 4ab + ac - 4bc]
  = 2a² - 8ab + 2ac - 8bc ✓

Respuesta: 2(a - 4b)(a + c)
```

**Rúbrica**:
- Extracción del factor numérico inicial: 20%
- Agrupación correcta: 30%
- Factorización completa: 35%
- Verificación: 15%

### Ejercicio 9
**Enunciado**: Factorizar $-5x^3y^2 + 15x^2y^3 - 10xy^4$

**Solución**:
```
PASO 1 - Extraer coeficiente negativo:
  Factor común: -5xy²

PASO 2 - Dividir:
  -5x³y² ÷ (-5xy²) = x²
  15x²y³ ÷ (-5xy²) = -3xy
  -10xy⁴ ÷ (-5xy²) = 2y²

PASO 3 - Factorización:
  -5xy²(x² - 3xy + 2y²)

PASO 4 - Factorizar el trinomio:
  x² - 3xy + 2y² = (x - y)(x - 2y)

RESPUESTA COMPLETA:
  -5xy²(x - y)(x - 2y)

Respuesta: -5xy²(x - y)(x - 2y)
```

**Rúbrica**:
- Extracción correcta con signo negativo: 25%
- Factorización del trinomio: 40%
- Respuesta completamente factorizada: 25%
- Verificación: 10%

### Ejercicio 10
**Enunciado**: Factorizar $(x + 3)^2 + 2(x + 3)$

**Solución**:
```
PASO 1 - Identificar expresión común:
  (x + 3) aparece en ambos términos

PASO 2 - Factor común: (x + 3)
  (x + 3)² + 2(x + 3) = (x + 3)[(x + 3) + 2]

PASO 3 - Simplificar:
  = (x + 3)(x + 3 + 2)
  = (x + 3)(x + 5)

VERIFICACIÓN:
  (x + 3)(x + 5) = x² + 5x + 3x + 15 = x² + 8x + 15
  (x + 3)² + 2(x + 3) = x² + 6x + 9 + 2x + 6 = x² + 8x + 15 ✓

Respuesta: (x + 3)(x + 5)
```

**Rúbrica**:
- Reconocimiento de expresión binomial común: 30%
- Extracción correcta: 30%
- Simplificación: 25%
- Verificación: 15%

### Ejercicio 12
**Enunciado**: Factorizar $a^2(b - c) + b^2(c - a) + c^2(a - b)$

**Solución**:
```
Esta es una expresión cíclica especial.

PASO 1 - Reescribir con signos:
  a²(b - c) + b²(c - a) + c²(a - b)
  = a²b - a²c + b²c - b²a + c²a - c²b

PASO 2 - Reagrupar:
  = a²b - b²a + b²c - c²b + c²a - a²c
  = ab(a - b) + bc(b - c) + ca(c - a)

PASO 3 - Propiedad cíclica:
  Esta expresión es CERO o factoriza como:
  = -(a - b)(b - c)(c - a)

Respuesta: -(a - b)(b - c)(c - a)
```

**Rúbrica**:
- Reconocimiento de patrón cíclico: 35%
- Manipulación algebraica correcta: 35%
- Factorización final: 20%
- Justificación: 10%

### Ejercicio 13
**Enunciado**: Resolver la ecuación $5x^2 - 15x = 0$ usando factor común

**Solución**:
```
PASO 1 - Factorizar por factor común:
  5x² - 15x = 0
  5x(x - 3) = 0

PASO 2 - Aplicar propiedad del producto cero:
  5x = 0  ó  x - 3 = 0

PASO 3 - Resolver cada ecuación:
  5x = 0  ⟹  x = 0
  x - 3 = 0  ⟹  x = 3

VERIFICACIÓN:
  Para x = 0: 5(0)² - 15(0) = 0 ✓
  Para x = 3: 5(3)² - 15(3) = 45 - 45 = 0 ✓

Respuesta: x = 0 o x = 3
```

**Rúbrica**:
- Factorización correcta: 30%
- Aplicación de producto cero: 25%
- Soluciones correctas: 30%
- Verificación: 15%

### Ejercicio 14
**Enunciado**: Factorizar $x^n(x^2 - 1) + x^{n+1}(x + 1)$

**Solución**:
```
PASO 1 - Identificar factor común:
  Factor mínimo en potencias: xⁿ

PASO 2 - Extraer xⁿ:
  = xⁿ[(x² - 1) + x(x + 1)]

PASO 3 - Simplificar dentro del corchete:
  = xⁿ[x² - 1 + x² + x]
  = xⁿ[2x² + x - 1]

PASO 4 - Factorizar el trinomio:
  2x² + x - 1 = (2x - 1)(x + 1)

RESPUESTA FINAL:
  xⁿ(2x - 1)(x + 1)

Respuesta: xⁿ(2x - 1)(x + 1)
```

**Rúbrica**:
- Manejo correcto de exponentes variables: 35%
- Simplificación algebraica: 30%
- Factorización del trinomio: 25%
- Respuesta completa: 10%

### Ejercicio 15 (Desafío)
**Enunciado**: Factorizar $a^4 + a^3b + a^2b^2 + ab^3$, luego usar el resultado para factorizar $x^4 + x^3 + x^2 + x$

**Solución**:
```
PARTE 1:
  a⁴ + a³b + a²b² + ab³
  
  Factor común: a
  = a(a³ + a²b + ab² + b³)
  
  Agrupar:
  = a[a²(a + b) + b²(a + b)]
  = a(a + b)(a² + b²)

PARTE 2:
  x⁴ + x³ + x² + x
  
  Observar: esto es a⁴ + a³b + a²b² + ab³ con a = x, b = 1
  
  Factor común: x
  = x(x³ + x² + x + 1)
  
  Agrupar:
  = x[x²(x + 1) + 1(x + 1)]
  = x(x + 1)(x² + 1)

Respuesta Parte 1: a(a + b)(a² + b²)
Respuesta Parte 2: x(x + 1)(x² + 1)
```

**Rúbrica**:
- Factorización Parte 1 con agrupación: 40%
- Conexión entre ambos problemas: 20%
- Factorización Parte 2: 30%
- Explicación completa: 10%
