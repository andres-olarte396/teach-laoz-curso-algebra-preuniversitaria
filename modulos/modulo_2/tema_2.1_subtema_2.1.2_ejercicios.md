# Ejercicios: Agrupación de términos

## Nivel Básico (60-74%)

### Ejercicio 1
**Enunciado**: Factorizar $ax + ay + bx + by$

**Solución**:
```
PASO 1 - Agrupar pares con factor común:
  Grupo 1: ax + ay → a(x + y)
  Grupo 2: bx + by → b(x + y)

PASO 2 - Observar el factor común (x + y):
  a(x + y) + b(x + y)

PASO 3 - Extraer (x + y):
  = (x + y)(a + b)

VERIFICACIÓN:
  (x + y)(a + b) = ax + ay + bx + by ✓

Respuesta: (x + y)(a + b)
```

**Rúbrica**:
- Agrupación correcta: 40%
- Identificación de factor común: 30%
- Factorización final: 30%

### Ejercicio 2
**Enunciado**: Factorizar $x^2 + 3x + 2x + 6$

**Solución**:
```
PASO 1 - Agrupar:
  (x² + 3x) + (2x + 6)

PASO 2 - Factor común en cada grupo:
  x(x + 3) + 2(x + 3)

PASO 3 - Extraer (x + 3):
  = (x + 3)(x + 2)

VERIFICACIÓN:
  (x + 3)(x + 2) = x² + 2x + 3x + 6 = x² + 5x + 6 ✓

Respuesta: (x + 3)(x + 2)
```

**Rúbrica**:
- Agrupación en pares: 35%
- Extracción de factores comunes: 35%
- Respuesta final correcta: 30%

```

  = (2m + 2n) - (m² + mn)

PASO 2 - Factor común en cada grupo:
  = 2(m + n) - m(m + n)

PASO 3 - Extraer (m + n):
  = (m + n)(2 - m)

VERIFICACIÓN:
  (m + n)(2 - m) = 2m - m² + 2n - mn ✓

Respuesta: (m + n)(2 - m)
```

**Rúbrica**:
- Reagrupación correcta con signos: 40%

### Ejercicio 7
**Enunciado**: Factorizar $a^3 + a^2 + a + 1$

**Solución**:
```
PASO 1 - Agrupar:
  (a³ + a²) + (a + 1)

PASO 2 - Factor común:
  = a²(a + 1) + 1(a + 1)

PASO 3 - Extraer (a + 1):
  = (a + 1)(a² + 1)

OBSERVACIÓN:
  (a² + 1) NO factoriza más en números reales

VERIFICACIÓN:
  (a + 1)(a² + 1) = a³ + a² + a + 1 ✓

Respuesta: (a + 1)(a² + 1)
```

**Rúbrica**:
- Agrupación correcta: 30%

### Ejercicio 8
**Enunciado**: Factorizar $x^3 - x^2 - x + 1$

**Solución**:
```
PASO 1 - Agrupar:
  (x³ - x²) + (-x + 1)
  = (x³ - x²) - (x - 1)

PASO 2 - Factor común:
  = x²(x - 1) - 1(x - 1)

PASO 3 - Extraer (x - 1):
  = (x - 1)(x² - 1)

PASO 4 - Factorizar diferencia de cuadrados:
  = (x - 1)(x - 1)(x + 1)
  = (x - 1)²(x + 1)

VERIFICACIÓN:
  (x - 1)²(x + 1) = (x² - 2x + 1)(x + 1)
  = x³ + x² - 2x² - 2x + x + 1
  = x³ - x² - x + 1 ✓

Respuesta: (x - 1)²(x + 1)
```

**Rúbrica**:
- Agrupación con signos negativos: 25%
- Primera factorización: 25%
- Factorización adicional de diferencia de cuadrados: 30%
- Verificación: 20%

### Ejercicio 9
**Enunciado**: Factorizar $2ax - 6bx + ay - 3by$

**Solución**:
```
OPCIÓN 1 - Agrupar por términos en x e y:
  (2ax - 6bx) + (ay - 3by)
  = 2x(a - 3b) + y(a - 3b)
  = (a - 3b)(2x + y)

OPCIÓN 2 - Agrupar por a y b:
  (2ax + ay) + (-6bx - 3by)
  = a(2x + y) - 3b(2x + y)
  = (2x + y)(a - 3b)

Ambas opciones son equivalentes

Respuesta: (a - 3b)(2x + y)
```

**Rúbrica**:
- Identificación de múltiples estrategias de agrupación: 30%
- Factorización correcta: 40%
- Reconocimiento de equivalencia: 20%
- Verificación: 10%

### Ejercicio 11
**Enunciado**: Factorizar $x^3 + 3x^2 - 4x - 12$

**Solución**:
```
PASO 1 - Agrupar:
  (x³ + 3x²) + (-4x - 12)

PASO 2 - Factor común:
  = x²(x + 3) - 4(x + 3)

PASO 3 - Extraer (x + 3):
  = (x + 3)(x² - 4)

PASO 4 - Factorizar diferencia de cuadrados:
  = (x + 3)(x - 2)(x + 2)

VERIFICACIÓN ESTRATÉGICA:
  Si x = -3: (-3)³ + 3(-3)² - 4(-3) - 12 = -27 + 27 + 12 - 12 = 0 ✓
  Si x = 2: 8 + 12 - 8 - 12 = 0 ✓

Respuesta: (x + 3)(x - 2)(x + 2)
```

**Rúbrica**:
- Agrupación correcta: 25%
- Primera factorización: 25%
- Factorización completa: 30%
- Verificación con raíces: 20%

### Ejercicio 13
**Enunciado**: Factorizar $am - bn + an - bm$

**Solución**:
```
ESTRATEGIA: Reordenar para agrupar mejor

PASO 1 - Reordenar:
  am + an - bm - bn

PASO 2 - Agrupar:
  (am + an) + (-bm - bn)
  = a(m + n) - b(m + n)

PASO 3 - Extraer (m + n):
  = (m + n)(a - b)

ALTERNATIVA (agrupación original):
  (am - bm) + (an - bn)
  = m(a - b) + n(a - b)
  = (a - b)(m + n)  [mismo resultado]

Respuesta: (m + n)(a - b)
```

**Rúbrica**:
- Reordenamiento estratégico: 30%
- Reconocimiento de múltiples caminos: 25%
- Factorización correcta: 30%
- Verificación de equivalencia: 15%

### Ejercicio 14
**Enunciado**: Resolver la ecuación $x^3 - 2x^2 - x + 2 = 0$ usando agrupación

**Solución**:
```
PASO 1 - Factorizar por agrupación:
  (x³ - 2x²) + (-x + 2) = 0
  x²(x - 2) - 1(x - 2) = 0
  (x - 2)(x² - 1) = 0

PASO 2 - Factorizar x² - 1:
  (x - 2)(x - 1)(x + 1) = 0

PASO 3 - Aplicar propiedad del producto cero:
  x - 2 = 0  ⟹  x = 2
  x - 1 = 0  ⟹  x = 1
  x + 1 = 0  ⟹  x = -1

VERIFICACIÓN para x = 2:
  2³ - 2(2²) - 2 + 2 = 8 - 8 - 2 + 2 = 0 ✓

Respuesta: x = {-1, 1, 2}
```

**Rúbrica**:
- Factorización por agrupación: 30%
- Factorización completa: 25%
- Soluciones correctas: 30%
- Verificación: 15%

### Ejercicio 15 (Desafío)
**Enunciado**: Factorizar $x^4 + x^3 + x^2 + x + xy^3 + y^3$

**Solución**:
```
Esta expresión requiere agrupación creativa.

PASO 1 - Observar la estructura:
  Términos con x: x⁴ + x³ + x² + x
  Términos con y³: xy³ + y³

PASO 2 - Factor común en el segundo grupo:
  = (x⁴ + x³ + x² + x) + y³(x + 1)

PASO 3 - Factorizar el primer grupo:
  x⁴ + x³ + x² + x
  = (x⁴ + x³) + (x² + x)
  = x³(x + 1) + x(x + 1)
  = (x + 1)(x³ + x)
  = (x + 1)·x(x² + 1)
  = x(x + 1)(x² + 1)

PASO 4 - Sustituir en la expresión:
  = x(x + 1)(x² + 1) + y³(x + 1)

PASO 5 - Factor común (x + 1):
  = (x + 1)[x(x² + 1) + y³]
  = (x + 1)(x³ + x + y³)

RESPUESTA FINAL:
  (x + 1)(x³ + x + y³)

Respuesta: (x + 1)(x³ + x + y³)
```

**Rúbrica**:
- Identificación de estructura compleja: 30%
- Múltiples niveles de agrupación: 35%
- Factorización completa: 25%
- Presentación clara: 10%
