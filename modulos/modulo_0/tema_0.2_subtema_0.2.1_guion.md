## DESARROLLO
Primero, definamos qué es.
Variable: Símbolo (letra) que representa un número desconocido o cambiante.

Veamos un ejemplo práctico:
Identificar elementos en 3x + 5.
La solución es sencilla si sigues el orden... Variable: x, Coeficiente: 3, Constante: 5.
