1. Comprender los conceptos clave de variables y constantes.

3. Evitar errores comunes mediante la práctica consciente.

## Contenido Teórico
### Conceptos Fundamentales
- Variable: Símbolo (letra) que representa un número desconocido o cambiante
- Constante: Valor fijo que no cambia
- Coeficiente: Factor numérico que multiplica a una variable
- Término algebriaco: Producto de variables y constantes

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: Identificar elementos en 3x + 5

**Solución**:
```
Paso 1. x es la letra -> Variable
Paso 2: 3 multiplica a x -> Coeficiente
Paso 3: 5 está solo -> Constante

Respuesta: Variable: x, Coeficiente: 3, Constante: 5
```

### Ejemplo 2

**Problema**: Elementos en fórmula A = πr²

**Solución**:
```
Paso 1. A y r pueden cambiar
Paso 2: π siempre vale ~3.1416

Respuesta: Variables: A, r; Constante: π
```


## Errores Comunes
❌ **Error 1**: Confundir 2x con 2+x
✅ **Correcto**: 2x significa 2 multiplicado por x

❌ **Error 2**: Pensar que x siempre vale lo mismo
✅ **Correcto**: x es una variable, su valor depende del contexto o problema

2. **Identificar patrones**: Busca estructuras conocidas.
