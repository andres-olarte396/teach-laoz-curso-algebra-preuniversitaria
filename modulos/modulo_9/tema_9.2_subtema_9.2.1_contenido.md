# Logaritmos

## Introducción
El estudio de **Logaritmos** es crucial para comprender el comportamiento de las funciones y sus aplicaciones. 

## Objetivos de Aprendizaje
1. Definir y explicar Logaritmos.
2. Aplicar propiedades para resolver problemas.
3. Analizar gráficas y comportamientos.

## Contenido Teórico
- y = log_b(x) <=> b^y = x
- log(ab) = log a + log b

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: log_2(8)

**Solución**:
```
Paso 1. 2^3 = 8

Respuesta: 3
```

### Ejemplo 2

**Problema**: Resolver 2^x = 16

**Solución**:
```
Paso 1. 2^x = 2^4 -> x=4

Respuesta: 4
```


## Errores Comunes
❌ **Error**: Confundir conceptos básicos.
✅ **Correcto**: Revisar definiciones cuidadosamente.

## Resumen
- Logaritmos permite modelar fenómenos variados.
- La práctica es fundamental.
