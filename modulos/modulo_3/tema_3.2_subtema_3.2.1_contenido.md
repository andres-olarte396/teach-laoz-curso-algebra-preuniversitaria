# Método de sustitución

## Introducción
**Método de sustitución** es un pilar fundamental en este módulo. Entender esto te permitirá conectar conceptos previos con nuevas herramientas algebraicas.

## Objetivos de Aprendizaje
1. Dominar el concepto de Método de sustitución.
2. Identificar patrones para resolución rápida.
3. Aplicar estas técnicas en ejercicios multinivel.

## Contenido Teórico
- Despejar una variable
- Sustituir en la otra ecuación

### Profundización
(Aquí se detalla la lógica interna de los conceptos presentados, conectando con Despejar una variable).

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: x+y=5; 2x-y=4

**Solución**:
```
Paso 1. De ec1. y=5-x
Paso 2: En ec2: 2x-(5-x)=4
Paso 3: 3x-5=4 -> 3x=9 -> x=3
Paso 4: y=5-3=2

Respuesta: x=3, y=2
```


## Errores Comunes
❌ **Error**: Saltarse pasos de verificación.
✅ **Correcto**: Comprobar siempre el resultado.

## Estrategias de Resolución
- Identifica la estructura del problema.
- Selecciona el método más eficiente (ej. factorización vs fórmula general).
- Verifica si la solución tiene sentido en el contexto.

## Aplicaciones
- Simplificación de expresiones en Física.
- Modelado de funciones de costos/ingresos.
- Base para el Cálculo Diferencial.
