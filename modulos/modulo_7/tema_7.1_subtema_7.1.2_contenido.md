# Función cuadrática

## Introducción
El estudio de **Función cuadrática** es crucial para comprender el comportamiento de las funciones y sus aplicaciones. 

## Objetivos de Aprendizaje
1. Definir y explicar Función cuadrática.
2. Aplicar propiedades para resolver problemas.
3. Analizar gráficas y comportamientos.

## Contenido Teórico
- Vértice h = -b/2a
- Eje de simetría
- Concavidad

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: Vértice de y = x² - 4x + 3

**Solución**:
```
Paso 1. h = -(-4)/2 = 2
Paso 2: k = f(2) = 4-8+3 = -1

Respuesta: (2, -1)
```

✅ **Correcto**: Revisar definiciones cuidadosamente.

## Resumen
- Función cuadrática permite modelar fenómenos variados.
- La práctica es fundamental.
