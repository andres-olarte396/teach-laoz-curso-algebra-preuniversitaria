# Suma y resta de expresiones algebraicas

## Introducción
**Suma y resta de expresiones algebraicas** es un tema fundamental en este curso. Dominar estos conceptos te permitirá avanzar hacia problemas más complejos con confianza.

1. Comprender los conceptos clave de suma y resta de expresiones algebraicas.
2. Resolver problemas aplicando las reglas y propiedades estudiadas.
3. Evitar errores comunes mediante la práctica consciente.

## Contenido Teórico
### Conceptos Fundamentales
- Términos semejantes: misma parte literal
- Coeficiente numérico y parte literal
- Reducción de términos semejantes

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: 3x + 5x - 2x

**Solución**:
```
Paso 1. (3 + 5 - 2)x
Paso 2: 6x

Respuesta: 6x
```

### Ejemplo 2

**Problema**: 2x² + 3x - 5x² + 7x

**Solución**:
```
Paso 1. 2x² - 5x² = -3x²
Paso 2: 3x + 7x = 10x
Paso 3: -3x² + 10x

Respuesta: -3x² + 10x
```

### Ejemplo 3

**Problema**: 4a + 3b - 2a + 5b

**Solución**:
```
Paso 1. 4a - 2a = 2a
Paso 2: 3b + 5b = 8b
Paso 3: 2a + 8b

Respuesta: 2a + 8b
```


## Errores Comunes
❌ **Error 1**: Sumar términos no semejantes: 2x + 3y = 5xy
✅ **Correcto**: No se pueden sumar: 2x + 3y

❌ **Error 2**: Ignorar signos: 3x - 5x = 8x
✅ **Correcto**: 3x - 5x = -2x


## Estrategias de Resolución
1. **Analizar el problema**: Identifica qué se pide y qué datos tienes.
2. **Identificar patrones**: Busca estructuras conocidas.
3. **Verificar paso a paso**: Revisa cada operación intermedia.
