## Ejemplos Resueltos
### Ejemplo 1

**Problema**: 12x⁵ ÷ 3x²

**Solución**:
```
Paso 1. 12 ÷ 3 = 4
Paso 2: x⁵ ÷ x² = x³
Paso 3: 4x³

Respuesta: 4x³
```

### Ejemplo 2

**Problema**: (x² + 5x + 6) ÷ (x + 2)

**Solución**:
```

Respuesta: x + 3
```

### Ejemplo 3

**Problema**: (2x³ - 3x² + 4x - 5) ÷ (x - 1)

**Solución**:
```

Respuesta: 2x² - x + 3, residuo: -2
```
