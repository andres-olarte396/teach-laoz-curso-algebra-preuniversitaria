## Ejemplos Resueltos
### Ejemplo 1

**Problema**: (x + 7)(x - 7)

**Solución**:
```
Paso 1. a² = x²
Paso 2: b² = 49
Paso 3: x² - 49

Respuesta: x² - 49
```

### Ejemplo 2

**Problema**: (3x + 2y)(3x - 2y)

**Solución**:
```
Paso 1. (3x)² - (2y)²
Paso 2: 9x² - 4y²

Respuesta: 9x² - 4y²
```

### Ejemplo 3

**Problema**: Factorizar: x² - 25

**Solución**:
```
Paso 1. √x² = x
Paso 2: √25 = 5
Paso 3: (x + 5)(x - 5)

Respuesta: (x + 5)(x - 5)
```
