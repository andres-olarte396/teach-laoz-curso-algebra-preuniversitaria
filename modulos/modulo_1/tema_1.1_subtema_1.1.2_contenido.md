# Multiplicación de expresiones

1. Comprender los conceptos clave de multiplicación de expresiones.

## Contenido Teórico
### Conceptos Fundamentales
- Propiedad distributiva: a(b + c) = ab + ac
- Multiplicación de monomios: (ax^m)(bx^n) = abx^(m+n)
- Multiplicación de polinomios
- Leyes de exponentes

## Ejemplos Resueltos
### Ejemplo 1

**Problema**: 3x × 4x²

**Solución**:
```
Paso 1. 3 × 4 = 12
Paso 2: x × x² = x³
Paso 3: 12x³

Respuesta: 12x³
```

### Ejemplo 2

**Problema**: 2x(3x + 5)

**Solución**:
```
Paso 1. 2x × 3x = 6x²
Paso 2: 2x × 5 = 10x
Paso 3: 6x² + 10x

Respuesta: 6x² + 10x
```

### Ejemplo 3

**Problema**: (x + 3)(x + 2)

**Solución**:
```
Paso 1. x×x = x²
Paso 2: x×2 + 3×x = 5x
Paso 3: 3×2 = 6
Paso 4: x² + 5x + 6

Respuesta: x² + 5x + 6
```
